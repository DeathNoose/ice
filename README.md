Основные возможности

- Покупка входных билетов** (фиксированная цена: 300₽)
- Бронирование коньков** (150₽/час) с возможностью выбора модели и размера
- Личный кабинет пользователя** с историей покупок и бронирований
- Административная панель** для управления коньками, просмотра бронирований и билетов
- Интеграция с ЮKassa** (тестовый режим)
- Адаптивный дизайн** для всех устройств
- Валидация форм** и маска для телефона
